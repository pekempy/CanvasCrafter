# CanvasCrafter

Premium graphic design editor in the browser.

## Features

- **Advanced Color System**: Custom HSV picker with **Brand Kits**.
- **Universal Eyedropper**: Precision pipette with real-time HUD (all browsers).
- **AI Tools**: Local background removal and edge tracing.
- **Layers & Typography**: Full hierarchy management and 3D text effects.
- **Export**: PNG, JPEG, and PDF support.

## Quick Start

```bash
npm install
npm run dev
```
